# Credit Risk Prediction Project

## Overview
This project contains the data analysis, modeling, and business case for the PRCL-0015 Credit Risk Prediction initiative.

## Files in this Repository
- **Business Case for PRCL-0015.pdf**: Documentation and details on the business case for the credit risk prediction model.
- **PRCL-0015_Credit_Risk_Prediction.ipynb**: Jupyter notebook containing the core credit risk prediction code.
- **Clint_project.ipynb**: Additional Jupyter notebook used for project analysis.
- **Untitled-1.py**: Python script for related tasks.
